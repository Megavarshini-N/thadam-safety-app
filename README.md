## Overview

🚨 Thadam Safety App

Thadam is a real-time public safety platform designed to make incident reporting faster, smarter, and more reliable.
It empowers citizens to report emergencies instantly while enabling authorities to monitor, verify, and respond efficiently through a centralized system.

With features like live alert tracking, location-based reporting, and streamlined communication, Thadam bridges the gap between the public and law enforcement — helping create safer, more responsive communities.
